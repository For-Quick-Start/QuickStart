## Tags

- Tags: <tagname>...</tagname>

- Headings: <h1>...</h1> (only one)
- Paragraphs: <p>...</p> (block level elements)

- Whitespaces in VSCodium and browser

- VSCodium: "lorem50", Alt+Z (text format)
- VSCodium: "p*3", "(p>lorem5)*3"

- Browser: development tools

- Italic: "<em>...</em>" - and not "<i>...</i>"
- Bold: "<strong>...</strong>" - and not "<b>...</b>"

## Lists

- '<ul>...</ul>', '<ol>...</ol>'
- '<li>...</li>'
- VSCodium: "ul>li*6"

## Images

- VSCodium: "img"
- '<img src="" alt="">' - we don't need to close images
- Relative paths: 'src', "images/profile.jpg", "./images/profile.jpg", "../images/profile.jpg"
- URL path (link)
- Accessibility: 'alt'
- Size: 'width="250" height="300"', Don't distort the aspect ration!
